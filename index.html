Breif Description

This game maker is designed to be an easy-to-use japanese style adventurous game maker. AVG game designers can use this maker by writing their own text files or implementing the GUI to develop their own AVG games even without any knowledge about programming. It is written in C++ based on a powerful 2D cross-platform open-source graphics library SFML.

Project Details

  • Language: C++
  • Tools: SFML
  • Platforms: Windows, MacOS X, Linux
  • Proceedings: Maker's GUI front
